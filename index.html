Create Bootable USB Drive

Steps to Disable Protection:

  1. Disable Virus & Threat Protection: Go to Windows Security → Virus & threat protection → Manage settings → Turn off Real-time protection, Cloud-delivered protection, and Automatic sample submission.
  2. Disable Core Isolation: Go to Windows Security → Device security → Core isolation details → Turn off Memory integrity.
  3. Disable Windows Defender & Updates (Using Tools):
    • • Extract the downloaded Defender Tools archive.
    • • Run Defender Control and click "Disable Windows Defender".
